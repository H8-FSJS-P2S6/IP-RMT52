# IP-RMT52

# Yugioh Card List Documentation

## Endpoints

List of available endpoints:

- POST /register
- POST /login
- POST /login/google

Routes below need authentication:

- GET /cards
- GET /cards/favorite
- GET /cards/favorite/:id
- GET /cards/:id
- POST /cards/favorite/add/:cardId
- PUT /cards/favorite/edit/:favoriteId
- DELETE /cards/favorite/delete/:favoriteId
- GET /archetype
- GET /randomcard
- GET /minigames

## 1. POST /register

Description:

- Register an account

Request:

- body

```json
{
  "username": "string",
  "email": "string",
  "password": "string"
}
```

```json
{
  "id": "number",
  "email": "string"
}
```

```json
{
  "message": "Validation error message"
}
```

```json

  "message": "Invalid token"
```
